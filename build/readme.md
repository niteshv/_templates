# Template

## Technology

- Webpack 4
- Scss
- Pug

## Installation

'npm install'

## Usage

### Build files

'npm run build'

### Start Development

'npm run start'

## Notes

- Dist folder is cleaned on build

## Minimum infomation to build

### config/setup.config.js

- Webpack paths, setup, pages and PWA are here

### icons

- favicon.ico
- src/img/icons/site-icon.png (square icon, will output correct sizes)
